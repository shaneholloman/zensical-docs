# Zensical documentation

This repository hosts the documentation for [Zensical]. If you find any issues
or have suggestions for improvement, please feel free to open an issue or
submit a pull request.

  [Zensical]: https://github.com/zensical/zensical

## Setup

Fork the repository, clone it, install Zensical, and run the preview server:

```
pip install zensical
zensical serve
```
